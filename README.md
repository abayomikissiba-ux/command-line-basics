# Command Line Basics - Git and GitHub Assignment

## Repository Initialization
git init  
Initialized the folder as a Git repository.

## Adding Files
git add .  
Added all project files to the staging area.

## First Commit
git commit -m "Initial commit"  
Saved the files to the local repository.

## Connecting to GitHub
git remote add origin https://github.com/abayomikissiba-ux/command-line-basics.git  
Linked the local repository to GitHub.

## Pushing to GitHub
git push -u origin main  
Uploaded the project to GitHub.

## Cloning the Repository
git clone https://github.com/abayomikissiba-ux/my-first-project.git  
Cloned the repository into a new folder.

## Adding a New File
Created goals.txt, committed, and pushed changes to GitHub.

## Pulling Changes
git pull origin main  
Merged remote changes with local repository.

## Description
This project demonstrates the basic Git and GitHub workflow including initialization, staging, committing, pushing, cloning, and merging changes.
